
In the Postman collection, all endpoints and features are configured with variables for easy testing. To get started, follow these steps to import the collection into Postman:

1. Download the Collection JSON:

- Download the Postman Collection JSON file in the same directory of this file.

2. Import the Collection:

- Open Postman and click on the "Import" button in the top left corner.
- Select the downloaded JSON file to import the collection into Postman.

3. Change the Admin Password:

- Once the collection is imported, locate the collection in the left sidebar.
- Click on the collection to open the collection window.
- Go to the Variables section in the navbar in the top.
- Locate the collection variable called admin_password.

4. Update the Admin Password:

- Click on the "Current value" of the variable.
- Enter the new password provided internally.

5. Save the Collection:

- Click on the "Save" button to save the updated collection with the new password.

6. Start Testing:

- You can now start testing all endpoints and features using the configured variables in the collection.

Note: Make sure to keep the updated collection file secure and do not share it publicly.
